# Triage Labels

工程技能会使用五个标准 triage 角色。此文件把这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串。

| mattpocock/skills 中的 label | 本仓库 tracker 中的 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待 reporter 补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已经足够明确，可交给 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个角色时，例如“应用 AFK-ready triage label”，使用上表中对应的本仓库 label 字符串。

如果将来 GitHub Issues 中使用了不同的标签命名，只需要编辑上表右侧一列。
